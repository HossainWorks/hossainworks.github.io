Pong

A classic Pong clone

Jun 01, 2023
1 minute read

Welcome to Pong, a duplicate of the classic Pong game! This is the first release of my Love2D game, and it contains many of the key features you’d expect in a Pong game, such as an opponent AI and a player pad that you can control with the W, S keys or the arrow keys.

Installation Guide

This port of Pong has release for Windows and Linux. But you can still play it on MacOS. (See how)

Windows Vasta or Newer

  • Download pong-win32.zip
  • Extract the zip and run Pong.exe

Linux

  • Download pong-x86_64.AppImage
  • Run the downloaded pong-x86_64.AppImage… Almost all linux distrobutions support AppImages.

MacOS 10.7

  • Download and extract the love-11.4-macos.zip
  • Navigate to the folder where love-11.4-macos.zip is extracted
  • Download pong.love and run it with the love executable
